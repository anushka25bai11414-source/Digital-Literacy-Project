## PROJECT OVERVIEW:
-This repository contains my Digital Literacy Project, which I completed for the CSE0001 course at VIT Bhopal University. 
-The project focused on my role as a Student Digital Ambassador. I explored, applied, and reflected on essential digital skills needed in academic and professional settings.
-The project includes five structured tasks, each linked to a specific course module. These tasks emphasize practical experience over theory. Through this project, I created digital content, built a professional online presence, practiced coding and collaboration tools, developed professional communication skills, and researched awareness of cybercrime.
-All tasks are organized in separate folders within this repository. You can also find supporting files, screenshots, and documentation here. The complete written report is available in the report folder.

This project covers the following key areas of Digital Literacy:

1.Understanding the concept and importance of digital literacy
2.Creating digital awareness content through infographic design
3.Building and maintaining a professional online portfolio
4.Practicing coding skills using structured online platforms
5.Using collaboration tools for academic data collection
6.Writing professional emails with proper etiquette
7.Understanding responsible social media usage
8.Researching UPI fraud and online financial safety
9.Learning safe digital payment practices and reporting methods
